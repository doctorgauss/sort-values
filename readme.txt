---------------------------------------------

Сортирует строковые данные и данные целого типа.
Данные должны быть записаны в файл.
Версия JRE - 1.8.

---------------------------------------------

Формат запуска программы из командной строки:
<путь к файлу для чтения "сырых" данных> <путь к файлу для сохранения сортированых данных> <тип данных> <тип сортировки>

Ключ для обозначения типа данных: i - целые числа, s - строки
Ключ для обозначения типа сортировки: a - по возрастанию, d - по убыванию

Примеры запуска:
java -jar sort-it.jar in.txt out.txt -i -a (для целых чисел по возрастанию)
java -jar sort-it.jar in.txt out.txt -i -d (для целых чисел по убыванию)
java -jar sort-it.jar in.txt out.txt -s -a (для строк по возрастанию)

---------------------------------------------

Собранный проект находится в файле sort-it.jar
При необходимости, для сборки проекта запустить скрипт:
- для Windows - assembly.bat
- для Linux - assembly.sh

Для тестового запуска программы к проекту приложены следующие файлы:
- start.bat/start.sh - скрипты запуска в среде Windows/Linux
- in.txt/out.txt - файл примера входных/выходных данных
